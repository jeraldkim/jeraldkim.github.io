Assignment

Note: Please review the lecture slides before you go through this assignment.

In this assignment, You will be creating your first website as well as answering some questions on design.

To start off, let's create a folder that will house all our files.

  1. Create a folder (call it whatever you want).
  2. Now open up your code editor, create a new file and save it in that folder as index.html. By default, a web browser will always open the index.html file in a folder if there are multiple html files.
  3. Go back into that folder and create another folder called 'assets'. To keep files organized, we like to make a separate folder specifically for non-html content like images.
  4. Within the assets folder, create another folder called 'img' and go place your favorite selfie in that folder.

Now that your workspace is all set up, you are ready to start making your very first website. If you double click on the index.html file right now, it should open a blank page on your default browser (we recommend Google Chrome). When you make changes to your code, you can refresh that page to see the changes.

Now try doing the following steps. You'll notice that we encourage you to google things!

  1. Write your name in huge, bold text. (Hint: look at the "Headings" slide)
  2. On the next line, write your major and year in italic text. (Hint: "Bold and italic text" slide)
  3. Add a paragraph about yourself. (Hint: "Text" slide)
  4. Create a bulleted list of your three favorite places to eat in Berkeley. (Hint: "Lists" slide)
  5. Add a picture of yourself so we can recognize you in class! (Hint: "Images" slide)
  6. Add a link that, when clicked, directs you to the your favorite website. The text within the link should be the name of the website. (Hint: "Links" slide)

Some common troubleshooting:

  • Images not showing up? Make sure that in the img tag, you match the image name exactly, including capitalization. .jpg and .JPG are not the same! We recommend keeping all image file names in lowercase for consistency. Make sure image types are also matched: .png and .jpg are not the same!
